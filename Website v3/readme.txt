Importing the database from SQL dump:
-Go to localhost/phpmyadmin
-Create a database with name "plcs"
-Go to the SQL tab
-Copy the contents of "plcs.sql" file into the SQL Query area
-Press Go
-Refresh the database
-The database is ready with the values! 

To run the website: 
Put the entire folder in xampp/htdocs/plcs
Type in the browser: localhost/plcs/index.php

The files will be updated time to time.
